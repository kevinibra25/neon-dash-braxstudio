**Neon Dash** is a high-speed, cyberpunk-themed endless runner built for the modern operative. Developed by **BraxStudio**, it combines minimalist neon aesthetics with precision arcade gameplay and a global competitive layer.

---

## ✨ Features

- **High-Octane Gameplay**: Simple but punishing mechanics. Jump or die.
- **Unique Player Identity**: The **Identity Verification System** ensures your handle is unique. Once you claim a name on the leaderboard, it belongs to your operative ID alone.
- **Global Leaderboard**: Compete with players worldwide for the top 5 spots in real-time.
- **Dynamic Difficulty**: The game gets harder as your score increases — new obstacle types, patterns, and challenges unlock at every tier.
- **Character Customization**: Choose your operative's color before entering the grid.
- **PWA Support**: Install directly on your phone from the browser — no app store needed.
- **Polished Visuals**:
  - Dynamic trail effects and particle systems.
  - Responsive canvas-based rendering.
  - Cyberpunk UI with "Neural Interface" styling.
- **Persistent Data**: Powered by Firebase Firestore for global rankings and score persistence.

- **Frontend**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/kevinibra25/neon-dash-braxstudio.git
   cd neon-dash-braxstudio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Environment Setup**
   Create a `.env` file in the root and add your Firebase credentials:
   ```env
   VITE_FIREBASE_API_KEY=your_key
   VITE_FIREBASE_AUTH_DOMAIN=your_domain
   VITE_FIREBASE_PROJECT_ID=your_id
   # ... add other firebase config variables
   ```

4. **Launch Dev Server**
   ```bash
   npm run dev
   ```
